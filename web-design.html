Proceso de desarrollo

1. Planificación y Diseño

Nos contactamos con el cliente para saber que requiere su proyecto (dominio, hosting, diseño páginas web, integración CMS, actualizaciones, mantenimiento, posicionamiento, etc.)

2. Recepción de Material

Pedimos al cliente los material disponible (logotipos, fotografías, videos, textos, etc.) para poder comenzar a diagramar el sitio web.

3. Estructura de la Web

Definimos estilo, tipografía, colores, páginas/secciones, menú, etc.

4. Gestor de contenidos

En caso de que el proyecto requiera un gestor de contenidos (CMS) se procederá a la instalación del sistema y bases de datos.

5. Presentación y Revisiones

Nos reunimos personalmente o de forma online con el cliente para presentar el diseño y hacer revisiones y/o modificaciones si es necesario.

6. Desarrollo

Una vez que el cliente está conforme con lo diseñado o con el tema comprado en caso de utilizar WordPress comenzamos a desarrollar todas las funciones de la web.

7. Revisiones y Correcciones

Previo al lanzamiento volvemos a revisarlo junto al cliente.

8. Soporte y Aprendizaje

En caso que se haya utilizado un CMS (gestor de contenidos) le enseñamos a utilizar las funciones de su web.

9. Entrega y Lanzamiento

Una vez que el cliente esté completamente conforme publicamos el sitio web online.
